PART 2 OVERVIEW (DRAFT)
========================

Goal
----
Part 2 evaluates advanced low-dimensional representations for single-cell data and tests whether diffusion-based denoising of VAE latent spaces improves downstream performance.
Two evaluation tracks are used:
1) cLISI-based embedding quality.
2) Label transfer quality (classification accuracy/F1 on held-out test cells).

Directory Structure
-------------------
- experiments/: Entry-point scripts for running experiments and plotting results.
- src/: Core implementation of models, datasets, configs, and pipeline orchestration.
- plotting/: Notebook-based figure generation for thesis-ready visualizations.

Core Pipeline
-------------
Main controller: src/pipeline.py

For each configuration (latent_dim, denoise_steps), Pipeline.run_single_experiment(...) does:
1) Load dataset from data/<data_type>.h5ad and split into train/val/test (stratified by cell_type when available).
2) Preprocess data and select highly variable genes (HVGs) from training data.
3) Train (or load cached) VAE in selected latent dimension.
4) Extract latent embeddings for train/val/test.
5) Train (or load cached) diffusion denoiser on training latents.
6) Denoise latent embeddings for train/val/test for chosen denoise_steps.
7) Compute PCA baseline on HVG features (fit on train, transform val/test).
8) Compute cLISI metrics on test representations.
9) Run label-transfer evaluation (default logistic regression) for HVG/PCA/VAE/Denoised features.
10) Return combined metrics and metadata.

The pipeline uses extensive caching in model_path to avoid retraining/recomputing:
- VAE model checkpoints and latent caches.
- Diffusion model checkpoints.
- Denoised latent caches (train/val/test).

Configuration Files
-------------------
1) src/config.py
- Defines dataclasses:
  - VAEConfig: architecture/training hyperparameters.
  - DiffusionConfig: diffusion schedule/model/training hyperparameters.
  - ExperimentConfig: dataset paths, grid values, seed, device.

2) src/utils/dataset.py
- Defines dataset wrappers:
  - scRNADataset for expression matrices.
  - LatentDataset for latent vectors.

3) src/utils/evaluation.py
- Contains utility function compute_clisi_metrics(...) for cLISI calculation on available embeddings.

Model Implementations
---------------------
1) src/models/vae.py
- Implements scRNAVAE (MLP encoder/decoder latent model).
- Includes vae_loss(...) = reconstruction loss + KL loss.
- Provides get_latent(...) helper for deterministic latent extraction.

2) src/models/diffusion.py
- Implements GaussianDiffusion with configurable beta schedules (linear/cosine/sigmoid).
- Implements denoising network LatentDenoiser.
- Supports reverse denoising from a chosen denoise_steps level.

Experiment Entry Points
-----------------------
1) experiments/run_grid_search.py
- Main large-scale runner across latent_dims x denoise_steps.
- Initializes Pipeline and iterates configurations.
- Writes incremental and final CSV result files.
- Writes consolidated train/val/test AnnData files containing generated representations.

2) experiments/run_single_experiment.py
- Runs one selected configuration through Pipeline.
- Useful for debugging or quick checks.

3) experiments/run_multiple_experiment.py
- Repeats best-found hyperparameter configuration for multiple random seeds/runs.
- Loads best hyperparameters from JSON and aggregates repeated-run results.

4) experiments/evaluate_label_tranfer.py
- Evaluates label transfer across stored representations from consolidated AnnData files.
- Trains classifier (default logistic regression; optional random forest path exists).
- Computes validation/test metrics: accuracy, weighted F1, macro F1, balanced accuracy.
- Saves intermediate/final CSV results.

Plotting Scripts (Python)
-------------------------
1) experiments/plot_results.py
- Plots cLISI-focused summaries from grid search CSV:
  - line plots, heatmaps, grouped comparisons, improvements, summary tables.

2) experiments/plot_label_transfer.py
- Plots label-transfer-focused summaries:
  - baseline comparisons, line plots, heatmaps, method comparisons,
    cLISI-vs-label-transfer correlations, improvement heatmaps, summary views.

How This Connects to the Thesis Question
----------------------------------------
Part 2 tests whether learned latent representations (VAE) and diffusion-denoised latent representations can better preserve biologically meaningful structure and improve annotation transfer compared with conventional baselines (HVG/PCA).
The combination of cLISI and label-transfer metrics quantifies both embedding-level and task-level performance across datasets and hyperparameter settings.
